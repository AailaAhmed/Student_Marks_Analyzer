# Student Marks Analyzer (Python)

This project analyzes students' marks using Python.
It demonstrates my learning progression from a basic script
to an interactive multi-subject analyzer.

## Versions Included

### 1. Non-Interactive Version
**File:** `marks_analyzer.py`
- Predefined student data
- Calculates highest, lowest, and average marks
- Suitable for understanding Python basics

### 2. Interactive Single-Subject Version
**File:** `interactive_marks_analyzer.py`
- Takes user input for students and marks
- Calculates class average
- Finds top-performing student(s)

### 3. Interactive Multi-Subject Version (Advanced)
**File:** `interactive_multi_subject_marks_analyzer.py`
- Supports multiple subjects
- Calculates:
  - Subject-wise averages
  - Overall class average
  - Class topper based on total marks
- Handles ties for top scores

## Technologies Used
- Python
- VS Code

## Purpose
This project was created as a learning exercise to strengthen
my understanding of Python dictionaries, loops, sorting, and logic building.
